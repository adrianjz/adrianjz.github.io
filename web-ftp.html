FTP Software

FTP stands for File Transfer Protocol and is used to transfer files between a client and a server. Usually signing in is done with a username and password, but FTP can be configured to allow anonymous connections. For more security, FTP can be secured with SSL/TLS, making the protocol FTPS, such as how securing HTTP with SSL/TLS makes it HTTPS.

There are many different FTP clients one can use. I personally have used FileZilla and enjoy it a lot because it is cross-platform and open source, so I can use it on pretty much any machine I use at home. Judging from the image for this topic, it appears that Rio might be a fan of Cyberduck, another free FTP client that has good support and is also released under the GPL license.

You should use a client that meets all of your needs. A good comparison is available on this Wikipedia page.

Even though FTP is widely supported, it was not intended to be secure, so it is not recommended for private data. All data transmission including files, usernames, passwords, and commands can be seen by anyone intercepting the network packets. Using FTPS or a different protocol such as SSH File Transfer Protocol will resolve the security issues.
