Select Your Editor

I had heard good things about both Sublime Text and Atom, as well as a few other editors, but these stood out to me because they were cross-platform. Since I usually use some version of Linux, cross-compatibility is important. In case I mess something up while experimenting in Linux, or if I just have to use a different workstation, I want to keep a consistent experience across operating systems as much as possible to work with my existing files. Then I saw that Sublime Text was proprietary and Atom was open-source, which made me lean more towards Atom. Both seem to be highly configurable, but Atom's is known as the "hackable text editor", and advertised that it was "hackable to the core—a tool you can customize to do anything but also use productively without ever touching a config file," which was immensely appealing to me. I like being able to tweak things to get them to my liking, so based on these preferences, I decided to go with Atom. To install Atom, I had to add a software repository and download the program. Not too hard. If anyone needs help on doing this in Linux, clearer instructions can be found here: http://tipsonubuntu.com/2016/08/05/install-atom-text-editor-ubuntu-16-04/ (Links to an external site.)

I'm enjoying Atom so far, so I think I'll stick with it!
