Styling Forms
Creating a Basic Form
Bootstrap has built-in styles for forms
Add label to input fields to match label to input
Clicking on label activates input field, important for mobile!
Also functional for screen readers
sr-only = hide labels except for screen-readers
add class="form-control" -> gives 100% width and highlights when active
Can wrap with form group class for better spacing
div class="form-group"
Place checkbox inside a div with a class of checkbox
wrap label around checkbox
For checkboxes and radio buttons, labels appear to right
Use pull-right class to keep button toward the right
Can also use buttons in addition to input types
Formatting Radio and Checkbox Controls
Semantic reason: checkboxes for 1+ options
Radio buttons for only one option
Radio buttons need to have same name to group them
Add a disabled state = different icon using bootstrap
class="radio-inline" to put buttons next to each other
Using Inline Forms
Styles apply for bigger than SM breakpoint
add class of "form-inline" to element wrapping the form
Can hide the labels to save space
Make sure you use sr-only labels for screen-reading devices
At smaller resolutions, forms show stacked anyway
text-align: center <- centers forms to middle of page
inline forms excellent for short set of inputs
Styling a Horizontal Form
Great for mobile users
Add class="form-horizontal" to elements
Tell label to have class of "col-sm-2"
Add div around input field, "col-sm-10" (add up to 12!)
Add class called control-label to get alignment to work
add form-group class to wrap checkboxes
Offset by some columns to align properly
Horizontal form takes effort, but works well on both mobile and desktop
Using Validation Styles
input type="text"
has-warning, has-error, has-success gives inputs colored borders
Must use control-label for label to modify label as well
Can add icons to validation for visual feedback
add "has-feedback" to control
span class="glyphicon glyphicon-ok form-control-feedback"
Icons are part of glyphicons library
Halflings are used, but glyphicon name used
Add aria-hidden="true" hides icon on screen readers
Add alternate feedback for screen readers
(success)
aria-describedby="nameStatus" ties span to feedback
