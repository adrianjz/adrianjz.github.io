HTML and Atom



Every webpage we visit has the same underlying code, HTML. This code is like the DNA of the web, and in order to create great pages, we must understand the fundamentals of HTML and start coding. Using an editor greatly enhances the web authoring experience, and the editor I focus on here is Atom. The content linked below explores how one can start writing HTML code from the very beginning.

A craftsman is only as good as his tools, so the importance of selecting a good editor cannot be stressed enough. Depending on your individual needs, one editor may suit you better than another, but it is a good idea to try out a few different ones to get a feel for them before building something large. Once you have mastered the basics of HTML, you can start creating pages on your own and improving your skillsets. Practice makes perfect!
