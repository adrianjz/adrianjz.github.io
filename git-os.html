Git for Your OS

So, Git was already installed on my machine, but I realized that it was not up to date. To get the most recent version of Git using a version of Ubuntu 16.04, I used the following three commands:

$ sudo add-apt-repository ppa:git-core/ppa $ sudo apt-get update $ sudo apt-get install git

What this did was install the Git repository to my system so that the package manager would be able to find the latest version. Then, I updated the package lists to update the available packages that can be downloaded. Finally, I installed git to get the newest version.

I use the command line a lot, but I decided to look for a command that could be useful that I didn't know about. I learned about the "find" command. If you need to look for a file but aren't sure what it was called, you can type just a few characters along with a wildcard to find all files matching that description.

For example, if I know that I have a file that starts with "dog", I can type "find dog*" and it will list all files that start with "dog" in that directory. If I want to find all of the JPEG images in a directory, I can type "find *.jpg"and it will list the images that match those characteristics. Could come in handy someday.

The command line is really useful once you get past the fear of using it, so I'd recommend everyone to try learning some commands for fun to get used to it!
